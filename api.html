Potatoblock HTTP API 使用指南

欢迎来到 Potatoblock 的开发文档页面！在这里，你将找到如何使用我们的 HTTP API 的详细说明。

API 概览

我们提供了多个 API 接口，以便用户进行数据存取、删除等操作。

1. 设置数据

请求方法：POST

请求路径：/set

请求头：

Content-Type: application/json
Authorization: Basic

请求体：

{
    "key": "YOUR_KEY",          // 替换为实际键
    "value": "YOUR_VALUE"       // 替换为实际值
}

响应示例：

HTTP/1.1 201 Created
Content-Type: application/json

{
    "message": "Data set successfully"
}

2. 获取数据

请求方法：GET

请求路径：/get?key=YOUR_KEY

请求头：

Authorization: Basic

响应示例：

HTTP/1.1 200 OK
Content-Type: application/json

{
    "YOUR_KEY": "YOUR_VALUE"   // 返回存储的值
}

3. 删除数据

请求方法：POST

请求路径：/delete

请求头：

Content-Type: application/json
Authorization: Basic

请求体：

{
    "key": "YOUR_KEY"           // 替换为要删除的键
}

响应示例：

HTTP/1.1 202 Accepted
Content-Type: application/json

{
    "message": "Data deletion successful"
}

4. 获取所有数据

请求方法：GET

请求路径：/data

请求头：

Authorization: Basic

响应示例：

HTTP/1.1 200 OK
Content-Type: application/json

{
    "key1": "value1",
    "key2": "value2"
}

5. 执行 SQL 查询

请求方法：POST

请求路径：/sql

请求头：

Content-Type: application/json
Authorization: Basic

请求体：

{
    "query": "YOUR_SQL_QUERY"   // 替换为实际的 SQL 查询
}

响应示例：

HTTP/1.1 200 OK
Content-Type: application/json

{
    "result": [   // 返回查询的结果
        ["row1_col1", "row1_col2"],
        ["row2_col1", "row2_col2"]
    ]
}
